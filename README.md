## Managing Unit

```console
curl -X GET --unix-socket /var/run/control.unit.sock  \
      http://localhost/control/applications/django/restart

curl -X PUT -d '"/var/log/access.log"' \
      --unix-socket /var/run/control.unit.sock \
      http://localhost/config/access_log
```
